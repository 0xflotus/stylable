Installation

Install Stylable as a dependency in your local project.

Using npm:

npm install stylable --save

Using yarn:

yarn add stylable

While the stylable package includes a programmatic API and can be used directly, we strongly suggest also installing the stylable-integration package that powers the integration of .st.css files in your projectâ€™s build cycle.
