Theme

Used in the top part of an application and to create a theme for a component library.

When importing a stylesheet use-st-theme directive to add its look and feel to your stylesheet. Any CSS definitions written inside the imported stylesheet will output to the final CSS.

@namespace "project";
:import {
    -st-theme: true;
    -st-from: './theme.st.css';
}

Override classes

Any shared classes and component variants defined at the theme level can be overridden in our stylesheet, by simply redefining them.

@namespace "project";
:import {
    -st-theme: true;
    -st-from: 'comp-lib/backoffice-theme.st.css';
    -st-named: emphasisBox, mainGallery;
}
.mainGallery { /* component variant extends Gallery */
    background: black;
}
.mainGallery::navButton {
    color: white;
}
.emphasisBox {
    border: 5px dashed black;
}

Override theme variables

Any variable defined in a theme file can be overridden. every CSS declaration that use that variable will be overridden under our stylesheet.

@namespace "project";
:import {
    -st-theme: true;
    -st-from: 'comp-lib/backoffice-theme.st.css';
    color1: black; /* override color1 with black */
    color2: purple; /* override color2 with purple */
}
