Stylable Themes Guide

What is a theme?

A theme is a way to implement and integrate a single “design system” into components from various sources.

A good theme is easily maintainable, and easily replaceable. It is created using a combination of best practices, and a set of Stylable features.

What makes a theme “good”?

  • easy to switch between different themes
  • all parts are easily customizable
